# Reference Verification Sources

Prefer these source types when checking whether references exist:

- DOI resolver or publisher page for journal articles.
- University repository records for academic articles when publisher access is limited.
- Library catalogue, publisher page, WorldCat, or ISBN records for books.
- Official course, university, or module material for internal teaching resources.

For author-date reference hygiene:

- Every bibliography item should normally have an in-text citation.
- Every in-text author-date citation should normally have a matching bibliography item.
- Course materials may be cited by institution name and year if no individual author is listed.
- If an item is only background reading, ask whether to keep it or label it as further reading.
